# &nbsp;　BOLT-Fix-UUIDs
BOLTの "MCIDを変更すると一部正常に動作しなくなるバグ" を直すデータパック。それ以上でも以下でもない。<br><br>

## ◈ 対応バージョン<br>
  - 1.16.x <br><br>
  
## ◈ 使い方
中身を解凍してdatapackフォルダ入れておけば、何もしなくても自動的に修復してくれます。<br>
正常に導入できているかは `/datapack list`　で確認することができます。<br><br>

## ◈ 注意
使う前に念の為バックアップをしておくことを推奨します。
